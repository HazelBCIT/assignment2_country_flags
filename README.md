# Installation
```bash
npm install --save-dev @playwright/test
```

```bash
npm run build
```

```bash
npm run start
```
## How to test
```bash
npm run test:e2e
# or
npx playwright test home.spec.ts
```
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
